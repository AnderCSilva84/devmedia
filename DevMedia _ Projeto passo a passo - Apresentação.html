Objetivo

A proposta desse projeto é criar um site sem estilização utilizando a tecnologia HTML. Teremos apenas um único arquivo HTML que conterá informações sobre um parque.

Ao criar o arquivo HTML desse projeto, você poderá colocar em prática o seguinte conceito:

  • Utilizar tags HTML (básicas)

Conheça melhor nosso projeto no flow abaixo:

#PraCegoVer - Transcrição do flow - Conhecendo o projeto
Nosso projeto é um site que contém informações sobre um parque. Esse site tem apenas um arquivo que contém diversas tags básicas do HTML, como a tag para o título, imagem, subtítulos e parágrafo.
Por se tratar da sua primeira missão criando um projeto, não vamos nos preocupar com a estética, ou seja, não vamos criar estilos CSS.
Nós vamos focar em posicionar e exibir os elementos HTML.

Para criar esse site, usamos algumas tags básicas do HTML, como vemos no flow a seguir:

#PraCegoVer - Transcrição do flow - Elementos utilizados no site
Nesse projeto utilizamos diversas tags básicas para criar o site.
Uma tag foi utilizada para representar o título do documento HTML (o nome que aparece na aba do navegador).
Um quadrado ao redor do título do conteúdo (Parque Estadual Ilha Bela) é usado para destacar. Além disso, uma imagem também foi usada no arquivo HTML.
Abaixo da imagem temos um subtítulo no conteúdo do arquivo. Também temos outros subtítulos no arquivo, só que esses com menor relevância.
Abaixo de cada subtítulo temos alguns elementos que representam parágrafos. Por fim, temos um link que foi adicionado no final do documento HTML.

O que vem agora?

Agora que você já conhece o projeto, o próximo passo pode ser visto no flow abaixo:

#PraCegoVer - Transcrição do flow - Agora é com você!
A sua missão é ganhar autonomia, fortalecer sua mente programadora e replicar esse projeto. Ele está dividido em passos para te orientar a construir cada parte do projeto.
Cada passo apresenta o conteúdo que você precisa estudar e uma orientação do que deve ser feito.
Não se preocupe!! Se você tiver dúvidas durante o processo, basta compartilhar com os nossos professores! Basta clicar em "Suporte ao aluno" e colocar sua dúvida na caixinha que nossos professores vão te atender.
Você também pode comparar a sua solução com a nossa clicando em "Ver Solução".
E lembre-se, o código do seu projeto não precisa estar 100% igual ao nosso. O importante é que o resultado final seja o mesmo, ou seja, que o software se comporte exatamente como esperado.
